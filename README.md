# Corporate Sales Intelligence & Demand Forecasting System 📊

An end-to-end data science and machine learning pipeline that analyzes historical operational sales data, detects systemic transaction anomalies, delivers predictive 3-month demand forecasts, and segments product portfolios for optimized inventory stocking strategies.

The final production models and strategy matrices are deployed live via an interactive, multi-page Streamlit web dashboard.

🔗 **[Live Interactive Dashboard Link](https://chappidilavanya02-web-end-to-end-sales-forecasting-d-app-favzzm.streamlit.app/)**

---

## 🚀 Key Project Capabilities

*   **Executive Sales Dashboard (Page 1):** Provides interactive high-level multi-year corporate performance tracking, annual fiscal breakdowns, and aggregated revenue KPIs.
*   **Predictive Demand Forecasting (Page 2):** Uses advanced time-series analysis to project weekly inventory requirements up to 24 weeks ahead, complete with an interactive risk-tolerance slider to calculate safety stock buffers.
*   **Machine Learning Market Segmentation (Page 3):** Automatically groups product sub-categories using a K-Means clustering algorithm based on sales volumes and market volatility to assign automated warehouse stocking rules.

---

## 🛠️ Data Infrastructure & Technical Stack

*   **Front-End UI Framework:** Streamlit (Multi-page configuration with custom CSS theme injections)
*   **Analytics & Visualizations:** Python, Pandas, Matplotlib
*   **Machine Learning Serialization:** Pickled data structures (`project_assets.pkl`)
*   **Data Models Referenced:** Time-series decomposition baseline and K-Means Segmentation Matrix
